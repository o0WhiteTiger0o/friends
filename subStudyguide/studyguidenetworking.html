STUDY GUIDE FOR NETWORKING

  • Compare physical topology with logical topology:

    - Physical topology mostly prefer to a network's hardware and how computer, other devices, and cables or radio signals work together to form the physical network. while logical topology mostly prefer to how software controls access to network resources and how specific resources such as applications and database are shared on the network.

  • Describe a client-server network model in contrast with a peer-to-peer(p2p) network model:

    Peer-to-peer (p2p) network model:

    - In a p2p network model, the operating system of each computer on the network is responsible for controlling access to its resources without centralize control. The computers, called nodes or hosts on the network, form a logical group of computers and users that share resources. Each computer on a p2p network controls its own administration, resources, and security.

    • Advantages and disadvantages of the p2p network model:
        • Advantages:
      1. They are simple to config.
      2. They are less expensive to setup and maintain
        Disadvantages:
      1. They are not scalable. (Up to 15 computers)
      2. They are not secure.
      3. They are not practical for connecting more than 15 computers because it's too time comsuming to manage the resources on the network.

    client-server network model:

    - In the client-server network model, resources are managed by the NOS(Network operating System) via a centralizedirectory database. The database can be managed by one or more servers, so long as they each have a similar NOS installed.

    - When windows server controls access to a group of computers, this group is called a windows domain

    - The centralize directory database that contains user account information and security for the entire group of computers is called AD(Active Directory).

    - A user can sign on to the network from any computer on the network to get the access to the resources that active directory allows. This process is managed by AD DS(Active Directory Domain Services)

    Advantages of client-server network model:

    - User account and passwords to the network are assigned in one place.

    - Access to multiple resources can be centrally granted to a single user or group of users.

    - Problems on the network can be monitored, diagnosed, and often fixed from one location.

    - Client-Server networks are more scalable than p2p networks.

  • Explain the role of AD(Active Directory) in a windows domain:

    The role of AD in a windows domain is monotored and managed the database for the entire domain.

  • Explain the purposes of various network services:

    • Web service: serves up webpages to clients. The primary protocol used by web servers and browsers(Clients) is HTTP(Hypertext Transfer Protocol). When HTTP is layered on top of an encryption protocol, such as SSL(Secure Sockets Layer) or TLS(Transport Layer Security), the result is HTTPS(HTTP Secure). which gives a secure transmission.

    • Email services: is a client-server application that involves two servers. The client uses SMTP(Simple Mail Transfer Protocol) to send an emails message to the first server, which is sometimes called the SMTP server. The first server sends the message on to the receiver's mail server, where it's stored until the recipient requests delivery. The recipient's mail server deliver's the message to the receiving client using one of two protocols: POP3(Post Office Protocol, version 3) or IMAP4(Internet Message Access Protocol, version 4).

    • DNS service:DNS(Domain Name System) helps clients find web servers over a network such as the Internet.

    • FTP service:is a client-server application that transfers file between two computers and it primarily uses FTP(File Transfer Protocol)

    • Database services: serve as a container for massive amounts of data that can be organized into tables and records.Users and applications can then access and interact with the data that is stored on a database server. DBMS(Database management system) is a software installed on the database server. It is responsible for making requested changes to the data and organizing the data for viewing, reporting, or exporting. Many DBMSs use the programing language such as SQL(Structured Query Language), Microsoft SQL Server, Oracle Database, and the open-source MySQL.

    • Remote access service:allows an administrator or other user to "remote in" to access a remote computer from the technician's local device to control the computer remotely. The Telnet protocol is a comman-line application included in many operating systems, but transmission is not encrypted. The SSH command in Linux uses the SSH(secure shell) protocol, which creates an encrypted channel or tunnel between two computers and provides security that Telnet lacks.In windows operating systems, RDP(remote desktop protocol) also provide secure, encrypted transmissions.
